# objectkind AGENTS

## 文件约定

- 运行 `just` 查看所有可用命令（Go 工具链：`just go ...`，示例：`just example ...`）
- 项目背景和模块索引见 `README.md`
- 行为约束见本文档

## 开发环境

- Go ≥ 1.26（见 `go.mod`）
- 无需额外初始化，直接 `just go test`、`just go gen` 等入口

## 编码风格

- 文档、注释、错误和测试说明使用中文，专有名词除外
- 先做最小盘点再开始改动
- 不做顺手重构、批量格式化、依赖升级或入口重排
- 不因目录已存在而默认新增模块、执行入口或注册关系
- 不直接编辑 `zz_generated.*` 等生成文件
- 能完成最小验证时先验证再交付；无法验证时说明缺口

## 什么时候停、什么时候问

- 需求边界不清且存在多种互斥实现时停下
- 需要新增模块、生成器或执行入口时先确认
- 需要改变根 justfile 入口分层或 CI 调用方式时先确认
- 需要回滚或覆盖用户已有变更时先确认
- 高风险操作、破坏性 git 操作和发布流程必须得到明确确认

## 规则生效

- 只有落盘到 `AGENTS.md` 的规则才生效
- 子目录 `AGENTS.md` 只覆盖确有必要的局部规则，未覆盖的继承上级
- 子目录 `internal/example/AGENTS.md` 对该 workspace 有局部覆盖
